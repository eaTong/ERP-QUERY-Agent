## 核心模块

### 1. 认证模块 (Auth)
- 用户名密码登录
- Session 管理
- 密码加密存储 (bcrypt)

### 2. 权限管理模块 (RBAC)
- 用户管理 (CRUD)
- 角色管理 (CRUD)
- 菜单管理 (CRUD)
- 用户-角色分配
- 角色-菜单分配

### 3. 数据源管理模块
- 外部数据源配置 (MySQL/SQL Server)
- 表映射配置
- 字段映射配置
- 查询规则配置
- 显示规则配置

### 4. AI 查询模块
- 自然语言转 SQL
- 表结构上下文注入
- 提示词规则应用
- SQL 执行与结果转换

## 技术选型理由

### 后端
- **Express**: 轻量、灵活、社区生态成熟
- **Prisma**: 类型安全的 ORM，开发体验好
- **Session**: 适合有状态的企业应用

### 前端
- **React + TypeScript**: 类型安全，开发效率高
- **Ant Design**: 组件丰富，企业级 UI
- **Zustand**: 轻量级状态管理，简单易用

## 安全措施

1. 密码加密存储 (bcrypt)
2. 外部数据源密码加密 (AES-256-CBC)
3. Session 认证
4. 请求日志记录
5. 统一的错误处理
